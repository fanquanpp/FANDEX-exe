# FANDEX 知识学习平台 - 离线版

## 快速开始

### Windows

1. 解压 zip 包到任意目录
2. 双击 `start.bat`
3. 浏览器自动打开，开始使用

### Linux / macOS

1. 解压 zip 包到任意目录
2. 终端执行：`chmod +x start.sh && ./start.sh`
3. 浏览器自动打开，开始使用

## 环境要求

- Node.js 22 或更高版本（仅用于启动本地服务器）
- 现代浏览器（Chrome / Edge / Firefox / Safari）

## 安装到桌面

打开后点击浏览器地址栏右侧的"安装"图标，可将 FANDEX 安装到桌面，后续无需通过浏览器访问。

## 离线说明

- 全部文档、搜索、术语表均可离线使用
- 阅读进度保存在本地浏览器中，不会上传到任何服务器

## 停止服务

- Windows：关闭 start.bat 窗口
- Linux / macOS：终端按 Ctrl+C

## 更新方式

下载最新版本的 zip 包，解压覆盖旧文件即可。进度数据保存在浏览器中，不会丢失。
