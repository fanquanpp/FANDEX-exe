## AI 智能体 术语表

## B

| 术语     | 英文                    | 释义                                                           |
| -------- | ----------------------- | -------------------------------------------------------------- |
| 标注     | Annotation              | 为训练数据添加标签或结构化信息的过程，是监督学习的基础环节     |
| 表征学习 | Representation Learning | 自动从原始数据中学习有效特征表示的方法，减少人工特征工程的依赖 |

## C

| 术语       | 英文              | 释义                                                                |
| ---------- | ----------------- | ------------------------------------------------------------------- |
| 上下文窗口 | Context Window    | 大语言模型单次推理能处理的最大 token 数量，决定输入输出的长度上限   |
| 采样策略   | Sampling Strategy | 控制模型生成文本时选择下一个 token 的策略，如贪心、束搜索、核采样等 |

## D

| 术语     | 英文       | 释义                                                                     |
| -------- | ---------- | ------------------------------------------------------------------------ |
| 多模态   | Multimodal | 能够同时处理和理解多种模态数据（文本、图像、音频、视频）的 AI 能力       |
| 代理循环 | Agent Loop | 智能体持续执行"感知-推理-行动"循环的运行模式，直至任务完成或达到终止条件 |

## E

| 术语 | 英文      | 释义                                                                         |
| ---- | --------- | ---------------------------------------------------------------------------- |
| 嵌入 | Embedding | 将高维离散数据映射为低维稠密向量的技术，使语义相近的数据在向量空间中距离更近 |

## F

| 术语             | 英文             | 释义                                                                 |
| ---------------- | ---------------- | -------------------------------------------------------------------- |
| Function Calling | Function Calling | 大模型根据用户意图生成结构化函数调用参数的能力，实现与外部工具的对接 |
| 反思             | Reflection       | 智能体评估自身输出和决策过程的能力，用于自我纠错和策略改进           |
| 泛化             | Generalization   | 模型在未见过的新数据上保持良好性能的能力，是衡量模型质量的关键指标   |

## G

| 术语     | 英文     | 释义                                                                |
| -------- | -------- | ------------------------------------------------------------------- |
| 工具使用 | Tool Use | 智能体调用外部工具（如搜索引擎、代码执行器、API）扩展自身能力的行为 |

## H

| 术语 | 英文          | 释义                                                 |
| ---- | ------------- | ---------------------------------------------------- |
| 幻觉 | Hallucination | 大语言模型生成看似合理但实际不正确或虚构内容的现象   |
| 回合 | Turn          | 智能体与用户或环境之间的一次交互轮次，包含输入和输出 |

## J

| 术语     | 英文             | 释义                                                       |
| -------- | ---------------- | ---------------------------------------------------------- |
| 记忆机制 | Memory Mechanism | 智能体存储和检索历史交互信息的能力，分为短期记忆和长期记忆 |

## K

| 术语         | 英文                           | 释义                                                                   |
| ------------ | ------------------------------ | ---------------------------------------------------------------------- |
| 检索增强生成 | Retrieval-Augmented Generation | 结合外部知识库检索与模型生成的技术，减少幻觉并提升回答的准确性和时效性 |

## M

| 术语     | 英文                   | 释义                                                                 |
| -------- | ---------------------- | -------------------------------------------------------------------- |
| MCP 协议 | Model Context Protocol | 标准化大模型与外部工具和数据源交互的开放协议，提供统一的工具调用接口 |
| 提示工程 | Prompt Engineering     | 设计和优化输入提示以引导大语言模型产生期望输出的技术和方法           |

## P

| 术语     | 英文      | 释义                                                       |
| -------- | --------- | ---------------------------------------------------------- |
| 评估基准 | Benchmark | 用于系统评估 AI 模型在特定任务上表现的标准数据集和评测方法 |

## Q

| 术语     | 英文                   | 释义                                                   |
| -------- | ---------------------- | ------------------------------------------------------ |
| 强化学习 | Reinforcement Learning | 通过与环境交互获得奖励信号来学习最优策略的机器学习范式 |

## R

| 术语  | 英文                           | 释义                                                             |
| ----- | ------------------------------ | ---------------------------------------------------------------- |
| ReAct | Reasoning + Acting             | 结合推理与行动的智能体框架，交替执行思考步骤和工具调用步骤       |
| RAG   | Retrieval-Augmented Generation | 检索增强生成，先从知识库检索相关文档再交由模型生成回答的技术架构 |

## S

| 术语   | 英文             | 释义                                                       |
| ------ | ---------------- | ---------------------------------------------------------- |
| 思维链 | Chain-of-Thought | 通过在提示中展示逐步推理过程来引导模型进行复杂推理的技术   |
| 水印   | Watermark        | 在模型生成内容中嵌入可检测的统计信号，用于识别 AI 生成内容 |

## T

| 术语  | 英文        | 释义                                                                               |
| ----- | ----------- | ---------------------------------------------------------------------------------- |
| Token | Token       | 大语言模型处理文本的最小单位，一个 token 约对应 0.75 个英文单词或 0.5-1 个中文字符 |
| 微调  | Fine-Tuning | 在预训练模型基础上使用特定领域数据进一步训练，使模型适应特定任务                   |

## W

| 术语       | 英文            | 释义                                                           |
| ---------- | --------------- | -------------------------------------------------------------- |
| 温度参数   | Temperature     | 控制模型生成随机性的超参数，值越高输出越多样，值越低输出越确定 |
| 向量数据库 | Vector Database | 专门存储和检索高维向量的数据库系统，支持高效的相似性搜索       |

## Z

| 术语   | 英文      | 释义                                                     |
| ------ | --------- | -------------------------------------------------------- |
| 智能体 | Agent     | 能够感知环境、自主决策并执行行动以实现目标的 AI 系统     |
| 对齐   | Alignment | 确保 AI 系统的行为与人类价值观和意图一致的技术和治理方法 |
| 规划   | Planning  | 智能体将复杂目标分解为可执行步骤序列的推理过程           |
