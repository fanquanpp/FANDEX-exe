## C# 术语表

## A

| 术语        | 英文        | 释义                                                                          |
| ----------- | ----------- | ----------------------------------------------------------------------------- |
| async/await | Async/Await | C# 异步编程的核心语法，async 标记异步方法，await 异步等待任务完成而不阻塞线程 |

## B

| 术语     | 英文            | 释义                                                           |
| -------- | --------------- | -------------------------------------------------------------- |
| 表达式树 | Expression Tree | 将代码表示为数据结构的树形对象，常用于 LINQ 提供程序的查询翻译 |

## C

| 术语   | 英文      | 释义                                                     |
| ------ | --------- | -------------------------------------------------------- |
| 程序集 | Assembly  | .NET 编译输出的基本部署单元，包含 IL 代码、元数据和资源  |
| 特性   | Attribute | 以声明方式为代码元素附加元数据的机制，运行时通过反射读取 |

## D

| 术语     | 英文            | 释义                                                                  |
| -------- | --------------- | --------------------------------------------------------------------- |
| 委托     | Delegate        | 类型安全的函数指针类型，定义方法签名，支持多播和异步调用              |
| 动态绑定 | Dynamic Binding | 使用 dynamic 类型在运行时而非编译时解析成员访问，支持与动态语言互操作 |

## E

| 术语 | 英文  | 释义                                                    |
| ---- | ----- | ------------------------------------------------------- |
| 事件 | Event | 基于委托的发布-订阅机制，允许对象在状态变化时通知订阅者 |

## F

| 术语     | 英文               | 释义                                                                            |
| -------- | ------------------ | ------------------------------------------------------------------------------- |
| 反射     | Reflection         | 在运行时检查和操作类型元数据的能力，支持动态创建实例和调用方法                  |
| 泛型约束 | Generic Constraint | 使用 where 子句限制泛型类型参数必须满足的条件，如实现特定接口或具有无参构造函数 |

## G

| 术语       | 英文     | 释义                                                               |
| ---------- | -------- | ------------------------------------------------------------------ |
| 感知迭代器 | Iterator | 使用 yield return 实现的延迟执行序列生成方法，编译器自动生成状态机 |

## H

| 术语   | 英文    | 释义                                                                                 |
| ------ | ------- | ------------------------------------------------------------------------------------ |
| 互操作 | Interop | C# 与非托管代码（如 Win32 API、COM 组件）交互的能力，通过 P/Invoke 和 COM 互操作实现 |

## J

| 术语     | 英文        | 释义                                                                          |
| -------- | ----------- | ----------------------------------------------------------------------------- |
| 记录类型 | Record Type | C# 9.0 引入的引用类型，基于值的相等性语义，自动生成 With 表达式支持和不可变性 |

## K

| 术语         | 英文                    | 释义                                                                               |
| ------------ | ----------------------- | ---------------------------------------------------------------------------------- |
| 可空引用类型 | Nullable Reference Type | C# 8.0 引入的编译期安全特性，区分可空与不可空引用类型以防止 NullReferenceException |

## L

| 术语 | 英文                      | 释义                                                                    |
| ---- | ------------------------- | ----------------------------------------------------------------------- |
| LINQ | Language Integrated Query | C# 内置的统一查询语法，支持对集合、数据库、XML 等数据源的声明式查询操作 |

## M

| 术语     | 英文             | 释义                                                           |
| -------- | ---------------- | -------------------------------------------------------------- |
| 模式匹配 | Pattern Matching | 使用 is、switch 表达式等对数据形状进行条件匹配和解构的语言特性 |

## P

| 术语     | 英文                | 释义                                                            |
| -------- | ------------------- | --------------------------------------------------------------- |
| 属性     | Property            | 封装字段的访问器机制，通过 get/set 访问器控制对私有字段的读写   |
| 顶级语句 | Top-Level Statement | C# 9.0 引入的简化程序入口写法，无需显式声明 Main 方法和命名空间 |

## R

| 术语     | 英文               | 释义                                                                |
| -------- | ------------------ | ------------------------------------------------------------------- |
| 垃圾回收 | Garbage Collection | .NET 运行时自动管理堆内存的机制，采用分代回收策略回收不再使用的对象 |

## S

| 术语             | 英文             | 释义                                                                      |
| ---------------- | ---------------- | ------------------------------------------------------------------------- |
| Source Generator | Source Generator | C# 9.0 引入的编译时代码生成器，在编译期间分析用户代码并自动生成附加源代码 |
| Span             | Span<T>          | 提供对连续内存区域的安全且零分配视图的类型，支持栈分配和托管/非托管内存   |

## T

| 术语   | 英文        | 释义                                                                  |
| ------ | ----------- | --------------------------------------------------------------------- |
| 值元组 | Value Tuple | C# 7.0 引入的值类型元组，支持命名元素和解构，比 System.Tuple 性能更优 |

## W

| 术语     | 英文             | 释义                                                                         |
| -------- | ---------------- | ---------------------------------------------------------------------------- |
| 扩展方法 | Extension Method | 在不修改原始类型的情况下向其添加方法的静态类语法糖，通过 this 修饰第一个参数 |

## Y

| 术语   | 英文      | 释义                                                              |
| ------ | --------- | ----------------------------------------------------------------- |
| 应用域 | AppDomain | .NET Framework 中的隔离边界，提供程序集加载和卸载的轻量级隔离单元 |
