## IoT 物联网 术语表

## A

| 术语 | 英文                        | 释义                                                           |
| ---- | --------------------------- | -------------------------------------------------------------- |
| ADC  | Analog-to-Digital Converter | 模数转换器，将模拟信号（如传感器电压）转换为数字信号的硬件模块 |
| ACK  | Acknowledgement             | 消息确认机制，接收方通知发送方消息已成功接收的应答信号         |

## B

| 术语     | 英文           | 释义                                                           |
| -------- | -------------- | -------------------------------------------------------------- |
| 边缘计算 | Edge Computing | 在靠近数据源的边缘节点执行计算任务，减少云端传输延迟和带宽消耗 |

## C

| 术语   | 英文   | 释义                                                       |
| ------ | ------ | ---------------------------------------------------------- |
| 传感器 | Sensor | 感知物理环境（温度、湿度、光照等）并将其转换为电信号的器件 |

## D

| 术语     | 英文                      | 释义                                                          |
| -------- | ------------------------- | ------------------------------------------------------------- |
| 低功耗   | Low Power                 | 设备在极低能耗下运行的设计目标，对电池供电的 IoT 设备至关重要 |
| 数字孪生 | Digital Twin              | 物理设备或系统的虚拟映射模型，用于仿真、监控和预测性维护      |
| 短距通信 | Short-Range Communication | 覆盖范围有限的无线通信技术，如蓝牙、Zigbee、Wi-Fi             |

## E

| 术语   | 英文            | 释义                                                       |
| ------ | --------------- | ---------------------------------------------------------- |
| 嵌入式 | Embedded System | 为特定控制功能设计的专用计算机系统，资源受限且实时性要求高 |

## F

| 术语     | 英文                 | 释义                                             |
| -------- | -------------------- | ------------------------------------------------ |
| 固件     | Firmware             | 嵌入式设备中存储在非易失性存储器上的底层控制软件 |
| 泛在计算 | Ubiquitous Computing | 计算能力无处不在地嵌入日常环境和物品中的计算范式 |

## G

| 术语 | 英文                         | 释义                                                                 |
| ---- | ---------------------------- | -------------------------------------------------------------------- |
| GPIO | General-Purpose Input/Output | 通用输入输出引脚，可通过软件配置为输入或输出模式，用于与外部设备交互 |
| 网关 | Gateway                      | 连接不同网络协议和通信标准的桥梁设备，实现 IoT 设备与云端的互通      |

## I

| 术语 | 英文                     | 释义                                                               |
| ---- | ------------------------ | ------------------------------------------------------------------ |
| I2C  | Inter-Integrated Circuit | 双线制串行通信协议，使用 SDA 数据线和 SCL 时钟线，支持一主多从拓扑 |

## J

| 术语       | 英文               | 释义                                                 |
| ---------- | ------------------ | ---------------------------------------------------- |
| 机对机通信 | Machine-to-Machine | 设备之间无需人工干预自动交换数据和执行操作的通信模式 |

## L

| 术语    | 英文                   | 释义                                                               |
| ------- | ---------------------- | ------------------------------------------------------------------ |
| LoRa    | Long Range             | 低功耗广域网通信技术，传输距离可达数公里，适用于远距离低频数据上报 |
| LoRaWAN | LoRa Wide Area Network | 基于 LoRa 物理层的媒体访问控制层协议，定义设备类别和网络架构       |

## M

| 术语 | 英文                                | 释义                                                             |
| ---- | ----------------------------------- | ---------------------------------------------------------------- |
| MQTT | Message Queuing Telemetry Transport | 轻量级发布/订阅消息协议，专为带宽有限和网络不稳定的 IoT 场景设计 |

## N

| 术语   | 英文           | 释义                                                                 |
| ------ | -------------- | -------------------------------------------------------------------- |
| NB-IoT | Narrowband IoT | 基于蜂窝网络的窄带物联网技术，覆盖广、功耗低，适用于地下和远距离场景 |

## O

| 术语     | 英文                | 释义                                                       |
| -------- | ------------------- | ---------------------------------------------------------- |
| OTA 更新 | Over-The-Air Update | 通过无线网络远程更新设备固件或软件的机制，无需物理接触设备 |

## P

| 术语 | 英文           | 释义                                                                      |
| ---- | -------------- | ------------------------------------------------------------------------- |
| 频段 | Frequency Band | 无线通信使用的频率范围，不同 IoT 技术工作在不同频段（如 Sub-GHz、2.4GHz） |

## R

| 术语 | 英文                       | 释义                                                            |
| ---- | -------------------------- | --------------------------------------------------------------- |
| RTOS | Real-Time Operating System | 实时操作系统，保证任务在确定时间内完成，广泛用于嵌入式 IoT 设备 |

## S

| 术语     | 英文                        | 释义                                                                       |
| -------- | --------------------------- | -------------------------------------------------------------------------- |
| SPI      | Serial Peripheral Interface | 高速同步串行通信协议，使用四线制（SCLK、MOSI、MISO、CS），适合短距高速传输 |
| 设备影子 | Device Shadow               | 云端维护的设备状态缓存，记录设备期望状态和实际状态，支持离线同步           |

## T

| 术语       | 英文                 | 释义                                                                        |
| ---------- | -------------------- | --------------------------------------------------------------------------- |
| 时序数据库 | Time Series Database | 专门存储和查询带时间戳数据的数据库，适合 IoT 传感器数据的持续写入和聚合查询 |

## U

| 术语 | 英文                                        | 释义                                                                   |
| ---- | ------------------------------------------- | ---------------------------------------------------------------------- |
| UART | Universal Asynchronous Receiver/Transmitter | 通用异步收发器，实现设备间串行通信的硬件协议，只需 TX 和 RX 两根信号线 |

## W

| 术语       | 英文         | 释义                                                     |
| ---------- | ------------ | -------------------------------------------------------- |
| 物联网平台 | IoT Platform | 提供设备接入、数据采集、规则引擎和应用使能的中间层云服务 |

## X

| 术语     | 英文                | 释义                                                         |
| -------- | ------------------- | ------------------------------------------------------------ |
| 协议转换 | Protocol Conversion | 将一种通信协议的数据格式转换为另一种，实现异构设备间的互操作 |

## Z

| 术语   | 英文     | 释义                                                                      |
| ------ | -------- | ------------------------------------------------------------------------- |
| Zigbee | Zigbee   | 基于 IEEE 802.15.4 的低功耗无线网状网络协议，适用于家庭自动化和传感器网络 |
| 执行器 | Actuator | 接收控制信号并执行物理动作（如开关阀门、驱动电机）的器件                  |
