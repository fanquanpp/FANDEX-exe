1. C# 语言概述

C#（读作 “C Sharp”）是由微软在 2000 年发布的现代、面向对象、类型安全的编程语言。它由 Anders Hejlsberg 领导设计，最初作为 .NET 战略的核心语言推出。C# 融合了 C/C++ 的强大能力和 Java 的简洁优雅，同时不断引入函数式编程、异步编程等现代范式。

1.1 核心特点

特点 描述 优势
类型安全 强类型系统，编译期检查类型兼容性 减少运行时错误，提高代码可靠性
面向对象 完整支持封装、继承、多态 代码复用性强，结构清晰
跨平台 基于 .NET 运行于 Windows/Linux/macOS 一次编写，多处运行
垃圾回收 自动内存管理，无需手动释放 减少内存泄漏风险
异步优先 async/await 原生语言级支持 简化异步编程模型
多范式 支持面向对象、函数式、泛型编程 灵活应对不同场景
持续演进 每年一个版本，快速迭代 始终保持语言现代化

2.2 .NET 8 新特性

  • 性能提升：JIT 编译优化、动态 PGO（Profile-Guided Optimization）默认启用
  • Native AOT 增强：更小的二进制体积、更好的反射支持
  • ASP.NET Core：Blazor SSR、新的身份认证 API
  • EF Core：复杂类型(Complex Types)、原始 SQL 查询增强
  • 新的数值类型：Int128、UInt128
// C# 12 主构造函数

    public Product GetProduct(int id)
    {
        logger.LogInformation("获取产品: {Id}", id);
        return repo.GetById<Product>(id);
    }

3. 开发环境搭建

3.1 安装 .NET SDK

# Windows - 使用 winget
winget install Microsoft.DotNet.SDK.9

# macOS - 使用 Homebrew
brew install dotnet-sdk

# Linux (Ubuntu)
sudo apt-get update && sudo apt-get install -y dotnet-sdk-9.0

# 验证安装
dotnet --version
dotnet --list-sdks

4. 项目类型与创建

4.1 常用项目模板

# 控制台应用
dotnet new console -n MyApp

# ASP.NET Core Web API
dotnet new webapi -n MyApi

# ASP.NET Core MVC
dotnet new mvc -n MyWeb

# Blazor Server / WebAssembly
dotnet new blazor -n MyBlazor

# 类库
dotnet new classlib -n MyLib

# Worker Service（后台服务）
dotnet new worker -n MyWorker

# xUnit 测试项目
dotnet new xunit -n MyTests

# MAUI 跨平台应用
dotnet new maui -n MyMauiApp

# 解决方案文件
dotnet new sln -n MySolution

4.2 项目结构

MyApp/
├── MyApp.csproj          # 项目文件（SDK 风格）
├── Program.cs            # 入口文件
├── appsettings.json      # 配置文件
├── GlobalUsings.cs       # 全局 using 声明
├── Models/               # 数据模型
├── Services/             # 业务逻辑
└── Controllers/          # 控制器（Web API）

4.3 csproj 文件（SDK 风格） 

<Project Sdk="Microsoft.NET.Sdk">
  <PropertyGroup>
    <TargetFramework>net9.0</TargetFramework>
    <Nullable>enable</Nullable>
    <ImplicitUsings>enable</ImplicitUsings>
    <RootNamespace>MyApp</RootNamespace>
    <Version>1.0.0</Version>
  </PropertyGroup>
  <ItemGroup>
    <PackageReference Include="Serilog" Version="4.2.0" />

  </ItemGroup>
</Project>

4.4 常用 CLI 命令

# 构建与运行
dotnet build                    # 构建项目
dotnet run                      # 运行项目
dotnet run --launch-profile Dev # 指定配置运行

# 包管理
dotnet add package Newtonsoft.Json    # 添加 NuGet 包
dotnet add package Serilog --version 4.2.0
dotnet remove package Newtonsoft.Json # 移除包
dotnet list package                   # 列出包
dotnet list package --outdated        # 检查过期包

# 发布
dotnet publish -c Release -o ./publish
dotnet publish -c Release -r linux-x64 --self-contained
dotnet publish -c Release -r win-x64 -p:PublishAot=true  # Native AOT

# 其他
dotnet test                     # 运行测试
dotnet format                   # 代码格式化
dotnet tool install -g dotnet-ef # 全局工具安装

5. Hello World

5.1 顶级语句（C# 9+）

// Program.cs - 最简形式
Console.WriteLine("Hello, World!");

// 使用变量和表达式
var name = "C# 开发者";
var version = Environment.Version;
Console.WriteLine($"你好, {name}! 当前 .NET 版本: {version}");

// 顶级语句中的方法
int Add(int a, int b) => a + b;
Console.WriteLine($"1 + 2 = {Add(1, 2)}");

// 异步入口
await Task.Delay(100);
Console.WriteLine("异步 Hello World!");

5.2 传统 Main 方法

using System;

namespace MyApp;

class Program
{
    static void Main(string[] args)
    {
        Console.WriteLine("Hello, World!");

        if (args.Length > 0)
        {
            Console.WriteLine($"参数: {string.Join(", ", args)}");
        }
    }
}

5.3 ASP.NET Core 最小 API 

var builder = WebApplication.CreateBuilder(args);
var app = builder.Build();

app.MapGet("/", () => "Hello, World!");
app.MapGet("/hello/{name}", (string name) => $"你好, {name}!");

app.Run();

6. .NET 版本选择指南

场景 推荐 .NET 版本 说明
新项目 .NET 9 (LTS) 最新长期支持版本
企业生产 .NET 8 (LTS) 稳定的长期支持版本
维护旧项目 .NET 6 (LTS) 旧版长期支持，2026年11月到期
游戏开发 .NET Standard 2.1 / .NET 8 取决于 Unity 版本
库开发 netstandard2.0 或 net8.0 最大化兼容性

提示：.NET 采用 STS（标准支持，18个月）和 LTS（长期支持，36个月）交替发布策略。.NET 8 和 .NET 9 均为重要版本，新项目建议直接使用 .NET 9。
