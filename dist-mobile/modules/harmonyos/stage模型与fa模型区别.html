HarmonyOS Stage模型与FA模型对比详解。

1. 两种模型对比

特性 FA 模型 Stage 模型
推荐度 旧版（不推荐） 新版（推荐）
开发语言 JS/Java ArkTS
UI 框架 JS UI ArkUI
应用模型 Ability UIAbility/ExtensionAbility
生命周期 Page 生命周期 UIAbility 生命周期
窗口管理 独立窗口 窗口复用
多实例 不支持 支持
跨设备 有限 原生支持

3. 迁移建议

  • 新项目直接使用 Stage 模型
  • FA 模型项目逐步迁移
  • HarmonyOS 4.0+ 推荐 Stage 模型
