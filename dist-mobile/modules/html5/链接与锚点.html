1.1 target 属性

值 行为
_self 当前窗口打开（默认）
_blank 新窗口/标签页打开
_parent 父框架中打开
_top 顶层窗口中打开 
<a href="https://example.com" target="_blank" rel="noopener noreferrer">外部链接</a>

安全提示：使用 target="_blank" 时务必添加 rel="noopener noreferrer"。

1.2 rel 属性

<a rel="noopener">无 opener</a>
<a rel="noreferrer">不发送 Referer</a>
<a rel="nofollow">不传递权重</a>
<a rel="ugc">用户生成内容</a>
