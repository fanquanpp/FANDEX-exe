1. 内积

1.1 定义

设 α=(a1​,a2​,…,an​)T，β=(b1​,b2​,…,bn​)T 是 Rn 中的向量，它们的内积（标准内积）定义为：

(α,β)=αTβ=a1​b1​+a2​b2​+⋯+an​bn​=∑i=1n​ai​bi​

1.2 内积的性质

  1. 对称性：(α,β)=(β,α)
  2. 线性性：(k1​α1​+k2​α2​,β)=k1​(α1​,β)+k2​(α2​,β)
  3. 正定性：(α,α)≥0，等号成立当且仅当 α=0

1.3 一般内积

更一般地，设 A 为 n 阶正定矩阵，可以定义加权内积：

(α,β)A​=αTAβ

1.4 Cauchy-Schwarz 不等式

∣(α,β)∣≤∥α∥⋅∥β∥

等号成立当且仅当 α 与 β 线性相关。

2. 向量的长度与距离

2.1 长度（范数）

向量 α 的长度（或范数）定义为：

∥α∥=(α,α)​=a12​+a22​+⋯+an2​​

2.2 长度的性质

  1. 非负性：∥α∥≥0，等号成立当且仅当 α=0
  2. 齐次性：∥kα∥=∣k∣⋅∥α∥
  3. 三角不等式：∥α+β∥≤∥α∥+∥β∥

2.3 单位向量

长度为 1 的向量称为单位向量。对非零向量 α，∥α∥α​ 是与 α 同方向的单位向量，称为 α 的单位化。

2.4 距离

两个向量 α 和 β 之间的距离定义为：

d(α,β)=∥α−β∥

2.5 夹角

两个非零向量 α 和 β 的夹角 θ 满足：

cosθ=∥α∥⋅∥β∥(α,β)​

3. 正交向量

3.1 定义

若 (α,β)=0，则称 α 与 β 正交，记作 α⊥β。

几何意义：两个向量正交意味着它们的夹角为 90°。

3.2 正交的性质

  1. 零向量与任何向量正交
  2. α⊥β⟺∥α+β∥2=∥α∥2+∥β∥2（勾股定理）
  3. 若 α⊥β1​ 且 α⊥β2​，则 α⊥(k1​β1​+k2​β2​)

3.3 正交向量组

若向量组 α1​,α2​,…,αs​ 中任意两个向量都正交（即 (αi​,αj​)=0，i=j），则称为正交向量组。

3.4 正交向量组的重要性质

定理：不含零向量的正交向量组一定线性无关。

证明：设 k1​α1​+k2​α2​+⋯+ks​αs​=0，两边与 αi​ 做内积：

ki​(αi​,αi​)=0

因 αi​=0，(αi​,αi​)>0，故 ki​=0。

3.5 标准正交向量组

若正交向量组中每个向量都是单位向量，则称为标准正交向量组（或规范正交组），即：

(αi​,αj​)=δij​={1,0,​i=ji=j​

标准正交基：若 α1​,…,αn​ 是 n 维向量空间的标准正交基，则任意向量 β 的坐标为：

xi​=(β,αi​)

即 β=∑i=1n​(β,αi​)αi​。

4. 正交补空间

4.1 定义

设 W 是 Rn 的子空间，所有与 W 中每个向量都正交的向量构成的集合称为 W 的正交补，记作 W⊥：

W⊥={α∈Rn∣(α,β)=0,∀β∈W}

4.2 性质

  1. W⊥ 也是子空间
  2. W∩W⊥={0}
  3. dim(W)+dim(W⊥)=n
  4. (W⊥)⊥=W
  5. Rn=W⊕W⊥（正交直和分解）

4.3 与矩阵的关系

  • N(A)=(Row(A))⊥
  • N(AT)=(Col(A))⊥

4.4 正交投影

向量 α 在子空间 W 上的正交投影 projW​α 满足：

α=projW​α+α⊥​

其中 α⊥​∈W⊥。

若 W 的一组标准正交基为 e1​,…,er​，则：

projW​α=∑i=1r​(α,ei​)ei​
