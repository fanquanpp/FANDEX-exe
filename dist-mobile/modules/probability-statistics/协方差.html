协方差

协方差的定义、性质、计算方法与协方差矩阵。

1. 协方差的定义

1.1 定义

设 X 和 Y 是两个随机变量，若 E[X−E(X)][Y−E(Y)] 存在，则称

Cov(X,Y)=E[X−E(X)][Y−E(Y)]

为 X 与 Y 的协方差。

1.2 协方差的计算公式

Cov(X,Y)=E(XY)−E(X)E(Y)

证明：

Cov(X,Y)=E[X−E(X)][Y−E(Y)]=E[XY−XE(Y)−YE(X)+E(X)E(Y)]

=E(XY)−E(X)E(Y)−E(Y)E(X)+E(X)E(Y)=E(XY)−E(X)E(Y)

1.3 协方差的直观意义

协方差衡量了两个随机变量的线性相关程度：

  • Cov(X,Y)>0：X 与 Y 正相关（X 增大时 Y 倾向于增大）
  • Cov(X,Y)<0：X 与 Y 负相关（X 增大时 Y 倾向于减小）
  • Cov(X,Y)=0：X 与 Y 不相关（无线性关系）

2. 协方差的性质

2.1 基本性质

  1. 对称性：Cov(X,Y)=Cov(Y,X)

  2. 自身协方差：Cov(X,X)=D(X)

  3. 常数：Cov(X,C)=0（C 为常数）

  4. 线性性：

Cov(aX,bY)=ab⋅Cov(X,Y)

Cov(X1​+X2​,Y)=Cov(X1​,Y)+Cov(X2​,Y)

  1. 更一般的双线性性：

Cov(∑i=1m​ai​Xi​,∑j=1n​bj​Yj​)=∑i=1m​∑j=1n​ai​bj​Cov(Xi​,Yj​)

  1. 方差与协方差的关系：

D(X±Y)=D(X)+D(Y)±2Cov(X,Y)

D(∑i=1n​Xi​)=∑i=1n​D(Xi​)+2∑i<j​Cov(Xi​,Xj​)

  1. 独立性推论：若 X 与 Y 独立，则 Cov(X,Y)=0

    注意：反之不成立，Cov(X,Y)=0 不能推出 X 与 Y 独立。

2.2 协方差的界

∣Cov(X,Y)∣≤D(X)⋅D(Y)​

这由柯西-施瓦茨不等式直接得到。

3. 协方差的计算

3.1 离散型

Cov(X,Y)=∑i=1∞​∑j=1∞​[xi​−E(X)][yj​−E(Y)]pij​

或

Cov(X,Y)=∑i=1∞​∑j=1∞​xi​yj​pij​−E(X)E(Y)

3.2 连续型

Cov(X,Y)=∫−∞+∞​∫−∞+∞​[x−E(X)][y−E(Y)]f(x,y)dxdy

或

Cov(X,Y)=∫−∞+∞​∫−∞+∞​xyf(x,y)dxdy−E(X)E(Y)

3.3 计算示例

例题：设 (X,Y) 的联合密度为

f(x,y)={2,0,​0<y<x<1其他​

求 Cov(X,Y)。

解：

E(X)=∫01​∫0x​2xdydx=∫01​2x2dx=32​

E(Y)=∫01​∫y1​2ydxdy=∫01​2y(1−y)dy=31​

E(XY)=∫01​∫0x​2xydydx=∫01​2x⋅2x2​dx=∫01​x3dx=41​

Cov(X,Y)=E(XY)−E(X)E(Y)=41​−32​×31​=41​−92​=361​

4. 不相关与独立

4.1 不相关的定义

若 Cov(X,Y)=0，即 E(XY)=E(X)E(Y)，则称 X 与 Y 不相关。

4.2 不相关与独立的关系

  • 独立 ⇒ 不相关
  • 不相关 ⇒ 独立

4.3 不相关但非独立的例子

设 X∼U(−1,1)，Y=X2，则

E(X)=0,E(XY)=E(X3)=0

Cov(X,Y)=E(XY)−E(X)E(Y)=0−0=0

X 与 Y 不相关，但 Y=X2，显然 X 与 Y 不独立。

4.4 特殊情况：二维正态分布

对于 (X,Y)∼N(μ1​,μ2​,σ12​,σ22​,ρ)：

X 与 Y 不相关⟺X 与 Y 独立⟺ρ=0

这是二维正态分布的特殊性质，一般分布不具备。

5. 协方差的应用

5.1 投资组合风险

D(wX+(1−w)Y)=w2D(X)+(1−w)2D(Y)+2w(1−w)Cov(X,Y)

当 Cov(X,Y)<0 时，可以通过分散投资降低风险。

5.2 回归分析

协方差是回归分析的基础，最小二乘回归系数为

β=D(X)Cov(X,Y)​
