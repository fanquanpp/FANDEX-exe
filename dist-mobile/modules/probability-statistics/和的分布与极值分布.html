1. 随机变量和的分布

1.1 离散型情形

设 X 和 Y 为独立离散型随机变量，分布律分别为 P(X=xi​) 和 P(Y=yj​)，则 Z=X+Y 的分布律为

P(Z=zk​)=∑xi​+yj​=zk​​P(X=xi​)P(Y=yj​)=∑i​P(X=xi​)P(Y=zk​−xi​)

1.2 连续型情形——卷积公式

设 X 和 Y 为独立连续型随机变量，密度函数分别为 fX​(x) 和 fY​(y)，则 Z=X+Y 的密度函数为

fZ​(z)=∫−∞+∞​fX​(x)fY​(z−x)dx=∫−∞+∞​fX​(z−y)fY​(y)dy

这称为卷积公式，记作 fZ​=fX​∗fY​。

1.3 卷积公式的推导

FZ​(z)=P(X+Y≤z)=∬x+y≤z​fX​(x)fY​(y)dxdy=∫−∞+∞​fX​(x)[∫−∞z−x​fY​(y)dy]dx

=∫−∞+∞​fX​(x)FY​(z−x)dx

fZ​(z)=dzd​FZ​(z)=∫−∞+∞​fX​(x)fY​(z−x)dx

1.4 卷积公式的应用

例题：设 X∼N(μ1​,σ12​)，Y∼N(μ2​,σ22​)，且 X 与 Y 独立，求 Z=X+Y 的分布。

解：利用卷积公式可以证明 Z∼N(μ1​+μ2​,σ12​+σ22​)。

更一般地，若 X1​,X2​,⋯,Xn​ 相互独立，Xi​∼N(μi​,σi2​)，则

∑i=1n​ai​Xi​∼N(∑i=1n​ai​μi​,∑i=1n​ai2​σi2​)

例题：设 X∼Exp(λ1​)，Y∼Exp(λ2​)，且 X 与 Y 独立（λ1​=λ2​），求 Z=X+Y 的密度。

解：

2. 随机变量差的分布

设 X 和 Y 独立，Z=X−Y，则

fZ​(z)=∫−∞+∞​fX​(x)fY​(x−z)dx=∫−∞+∞​fX​(y+z)fY​(y)dy

3. 随机变量商的分布

3.1 商的密度公式

设 X 和 Y 为独立连续型随机变量，Z=YX​，则

fZ​(z)=∫−∞+∞​∣y∣fX​(zy)fY​(y)dy

3.2 推导

FZ​(z)=P(YX​≤z)=∬x/y≤z​fX​(x)fY​(y)dxdy

=∫0+∞​FX​(zy)fY​(y)dy+∫−∞0​[1−FX​(zy)]fY​(y)dy

对 z 求导即得商的密度公式。

4. 最大值分布

4.1 公式

设 X1​,X2​,⋯,Xn​ 相互独立，分布函数分别为 F1​,F2​,⋯,Fn​，则 M=max(X1​,X2​,⋯,Xn​) 的分布函数为

FM​(x)=P(M≤x)=P(X1​≤x,X2​≤x,⋯,Xn​≤x)=∏i=1n​Fi​(x)

若 X1​,X2​,⋯,Xn​ 独立同分布，分布函数为 F(x)，则

FM​(x)=[F(x)]n

4.2 密度函数

若 X1​,X2​,⋯,Xn​ 独立同分布，密度为 f(x)，则

fM​(x)=n[F(x)]n−1f(x)

5. 最小值分布

5.1 公式

设 X1​,X2​,⋯,Xn​ 相互独立，分布函数分别为 F1​,F2​,⋯,Fn​，则 N=min(X1​,X2​,⋯,Xn​) 的分布函数为

FN​(x)=1−P(N>x)=1−∏i=1n​P(Xi​>x)=1−∏i=1n​[1−Fi​(x)]

若 X1​,X2​,⋯,Xn​ 独立同分布，分布函数为 F(x)，则

FN​(x)=1−[1−F(x)]n

5.2 密度函数

若 X1​,X2​,⋯,Xn​ 独立同分布，密度为 f(x)，则

fN​(x)=n[1−F(x)]n−1f(x)

6. 极值分布的应用

6.1 系统可靠性

串联系统：n 个元件串联，系统寿命 T=min(T1​,T2​,⋯,Tn​)

并联系统：n 个元件并联，系统寿命 T=max(T1​,T2​,⋯,Tn​)

6.2 指数分布的极值

设 X1​,X2​,⋯,Xn​ 独立同分布，Xi​∼Exp(λ)，则

Fmin​(x)=1−e−nλx,x>0

即 min(X1​,⋯,Xn​)∼Exp(nλ)。

6.3 均匀分布的极值

设 X1​,X2​,⋯,Xn​ 独立同分布，Xi​∼U(0,1)，则

fmax​(x)=nxn−1,0<x<1

fmin​(x)=n(1−x)n−1,0<x<1
