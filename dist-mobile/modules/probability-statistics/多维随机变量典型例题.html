1. 联合分布与边缘分布

2. 条件分布

3. 独立性判定

例题5

设 (X,Y) 的联合密度为

f(x,y)=⎩⎨⎧​23​x,0,​0<x<1,−x<y<x其他​

判断 X 与 Y 是否独立。

解：非零区域 {(x,y):0<x<1,−x<y<x} 不是矩形，故 X 与 Y 不独立。

验证：

fX​(x)=∫−xx​23​xdy=3x2,0<x<1

fY​(y)=∫∣y∣1​23​xdx=43​(1−y2),−1<y<1

fX​(x)fY​(y)=3x2⋅43​(1−y2)=49x2(1−y2)​=23​x

4. 和的分布

例题7

设 X∼U(0,1)，Y∼U(0,1)，且 X 与 Y 独立，求 Z=X+Y 的密度。

解：

fZ​(z)=∫−∞+∞​fX​(x)fY​(z−x)dx

fZ​(z)=⎩⎨⎧​z,2−z,0,​0<z<11≤z<2其他​

5. 极值分布

P(T>20)=1−P(T≤20)=1−[P(T1​≤20)]5=1−(1−e−2)5

6. 综合题

例题11

设 (X,Y) 的联合密度为

f(x,y)={2,0,​0<y<x<1其他​

求 Z=X−Y 的密度函数。

解：Z 的取值范围为 (0,1)。

FZ​(z)=P(X−Y≤z)=∬x−y≤z​f(x,y)dxdy

当 0<z<1 时：

P(X−Y>z)=∫z1​∫0x−z​2dydx=∫z1​2(x−z)dx=(1−z)2

FZ​(z)=1−(1−z)2

fZ​(z)=2(1−z),0<z<1

例题12

设 X 与 Y 独立，X∼Exp(1)，Y∼Exp(1)，求 Z=X+YX​ 的分布。

解：令 U=X+Y，V=X+YX​，则 X=UV，Y=U(1−V)。

Jacobian 行列式为 ∣J∣=u。

联合密度：

fU,V​(u,v)=fX​(uv)fY​(u(1−v))⋅u=e−uv⋅e−u(1−v)⋅u=ue−u,u>0,0<v<1

fV​(v)=∫0+∞​ue−udu=1,0<v<1

故 Z∼U(0,1)。

f(x,y)=⎩⎨⎧​π1​,0,​x2+y2≤1其他​

故 X 与 Y 不独立。
