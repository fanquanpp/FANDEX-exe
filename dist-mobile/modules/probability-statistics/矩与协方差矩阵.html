1. 矩的概念

1.1 原点矩

设 X 为随机变量，k 为正整数，若 E(Xk) 存在，则称

μk​=E(Xk)

为 X 的 k 阶原点矩。

特别地，一阶原点矩 μ1​=E(X) 即为数学期望。

1.2 中心矩

设 X 为随机变量，k 为正整数，若 E[X−E(X)]k 存在，则称

νk​=E[X−E(X)]k

为 X 的 k 阶中心矩。

特别地：

  • 一阶中心矩 ν1​=0
  • 二阶中心矩 ν2​=D(X)（方差）

1.4 混合矩

设 X,Y 为随机变量，k,l 为非负整数，若 E(XkYl) 存在，则称

μk,l​=E(XkYl)

为 X 与 Y 的 k+l 阶混合原点矩。

类似地，混合中心矩为

νk,l​=E[X−E(X)]k[Y−E(Y)]l

特别地，ν1,1​=Cov(X,Y)。

2. 偏度与峰度

2.1 偏度

γ1​=ν23/2​ν3​​=[D(X)]3/2E[X−E(X)]3​

偏度衡量分布的不对称性：

  • γ1​>0：右偏（正偏），分布右侧尾部较长
  • γ1​<0：左偏（负偏），分布左侧尾部较长
  • γ1​=0：对称分布

正态分布的偏度为 0。

2.2 峰度

γ2​=ν22​ν4​​−3=[D(X)]2E[X−E(X)]4​−3

峰度衡量分布的尖峰程度（相对于正态分布）：

  • γ2​>0：尖峰分布（比正态分布更尖）
  • γ2​<0：平坦分布（比正态分布更平）
  • γ2​=0：与正态分布相同

正态分布的峰度为 0（因为 ν22​ν4​​=3）。

3. 协方差矩阵

3.1 定义

设 n 维随机变量 X=(X1​,X2​,⋯,Xn​)T，令
