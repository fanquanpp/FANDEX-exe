随机变量的独立性

6 min Intermediate 2026/6/14

二维随机变量独立性的定义、判定方法、独立性的性质与应用。

1. 随机变量独立性的定义

1.1 定义

设 (X,Y) 是二维随机变量，若对任意实数 x,y，都有

F(x,y)=FX​(x)⋅FY​(y)

即

P(X≤x,Y≤y)=P(X≤x)⋅P(Y≤y)

则称随机变量 X 与 Y 相互独立。

1.2 等价定义

离散型：X 与 Y 独立 ⟺ 对所有 i,j，

P(X=xi​,Y=yj​)=P(X=xi​)⋅P(Y=yj​)

即 pij​=pi⋅​⋅p⋅j​

连续型：X 与 Y 独立 ⟺ 在 f(x,y) 的连续点处，

f(x,y)=fX​(x)⋅fY​(y)

2. 独立性的判定方法

2.1 定义法

直接验证联合分布等于边缘分布的乘积。

2.2 分解法

若联合密度 f(x,y) 可以分解为

f(x,y)=g(x)⋅h(y)

其中 g(x) 只与 x 有关，h(y) 只与 y 有关，且 f(x,y) 的非零区域为矩形（即 x 和 y 的取值范围互不依赖），则 X 与 Y 独立。

注意：非零区域必须是矩形区域，否则即使密度可分解，也不独立。

2.3 判定步骤

  1. 求出联合分布（分布律或密度）
  2. 求出边缘分布
  3. 检验联合分布是否等于边缘分布的乘积

3. 独立性的重要性质

3.1 独立变量的函数

若 X 与 Y 独立，则 g(X) 与 h(Y) 也独立，其中 g 和 h 为连续函数。

3.2 独立变量的概率

若 X 与 Y 独立，则对任意 Borel 集 A,B，

P(X∈A,Y∈B)=P(X∈A)⋅P(Y∈B)

3.3 独立变量的期望

若 X 与 Y 独立且期望存在，则

E(XY)=E(X)⋅E(Y)

3.4 独立变量的方差

若 X 与 Y 独立且方差存在，则

D(X+Y)=D(X)+D(Y)

D(X−Y)=D(X)+D(Y)

4. 独立性判定的典型例题

例题1

设 (X,Y) 的联合密度为

f(x,y)={4xy,0,​0<x<1,0<y<1其他​

fX​(x)⋅fY​(y)=4xy=f(x,y)✓

且非零区域为矩形，故 X 与 Y 独立。

例题2

f(x,y)={2,0,​0<x<y<1其他​

解：非零区域为三角形 {(x,y):0<x<y<1}，不是矩形，故 X 与 Y 不独立。

也可以验证：

fX​(x)=2(1−x),fY​(y)=2y

5. 多个随机变量的独立性

5.1 定义

设 X1​,X2​,⋯,Xn​ 是 n 个随机变量，若对任意实数 x1​,x2​,⋯,xn​，有

F(x1​,x2​,⋯,xn​)=FX1​​(x1​)⋅FX2​​(x2​)⋯FXn​​(xn​)

则称 X1​,X2​,⋯,Xn​ 相互独立。

5.2 独立随机变量和的性质

若 X1​,X2​,⋯,Xn​ 相互独立，则

  1. D(∑i=1n​Xi​)=∑i=1n​D(Xi​)

  2. D(∑i=1n​ai​Xi​)=∑i=1n​ai2​D(Xi​)

  3. 独立正态变量的线性组合仍为正态变量

5.3 独立与两两独立

  • 相互独立 ⇒ 两两独立
  • 两两独立 ⇒ 相互独立

这与事件的独立性类似。
