## 选择题

### 1. 以下关于操作系统的说法，正确的是？

A. 进程是资源分配的最小单位，线程是 CPU 调度的最小单位
B. 线程是资源分配的最小单位，进程是 CPU 调度的最小单位
C. 进程和线程都是资源分配的最小单位
D. 进程和线程都是 CPU 调度的最小单位

<details>
<summary>查看答案</summary>
**答案**: A
**解析**: 进程拥有独立的地址空间和资源，是操作系统资源分配的基本单位。线程共享进程的资源，仅拥有独立的栈和寄存器，是 CPU 调度的基本单位。
</details>

### 2. 一个 32 位地址总线的系统，最大可寻址空间是多少？

A. 2^16 B
B. 2^24 B
C. 2^32 B = 4 GB
D. 2^64 B

<details>
<summary>查看答案</summary>
**答案**: C
**解析**: n 位地址总线可寻址 2^n 个地址，每个地址对应 1 字节。32 位地址总线可寻址 2^32 = 4,294,967,296 字节 = 4 GB。
</details>

### 3. TCP 三次握手的正确顺序是？

A. SYN -> ACK -> SYN+ACK
B. SYN -> SYN+ACK -> ACK
C. ACK -> SYN -> SYN+ACK
D. SYN+ACK -> SYN -> ACK

<details>
<summary>查看答案</summary>
**答案**: B
**解析**: 客户端先发 SYN，服务器回 SYN+ACK，客户端再发 ACK，连接建立。三次握手确保双方都能收发数据，防止历史连接请求导致资源浪费。
</details>

### 4. 关于进程间通信 (IPC)，以下哪种方式不需要内核参与？

A. 管道 (Pipe)
B. 共享内存 (Shared Memory)
C. 消息队列 (Message Queue)
D. 信号量 (Semaphore)

<details>
<summary>查看答案</summary>
**答案**: B
**解析**: 共享内存是最快的 IPC 方式，进程直接读写同一块内存区域，无需内核在进程间拷贝数据。但需要配合信号量等同步机制使用。管道、消息队列和信号量都需要系统调用，涉及内核参与。
</details>

### 5. 以下哪种页面置换算法可能产生 Belady 异常？

A. OPT (最优置换)
B. LRU (最近最少使用)
C. FIFO (先进先出)
D. Clock (时钟算法)

<details>
<summary>查看答案</summary>
**答案**: C
**解析**: Belady 异常是指分配更多物理页框反而导致缺页率上升的现象。FIFO 算法可能出现此异常，而 LRU 和 OPT 属于栈式算法，不会产生 Belady 异常。
</details>

## 简答题

### 1. 解释死锁的四个必要条件，并给出预防策略

**参考答案**:

| 条件       | 含义                       | 预防策略                 |
| ---------- | -------------------------- | ------------------------ |
| 互斥       | 资源一次只能被一个进程使用 | 将共享资源改为可同时访问 |
| 占有并等待 | 持有资源的同时请求新资源   | 一次性申请所有资源       |
| 不可抢占   | 已获得的资源不能被强行回收 | 允许抢占（回滚机制）     |
| 循环等待   | 进程间形成环形等待链       | 对资源编号，按序申请     |

### 2. 简述 HTTP/1.1 与 HTTP/2 的主要区别

**参考答案**:

- **多路复用**: HTTP/2 在同一 TCP 连接上并行多个请求/响应，HTTP/1.1 需要管线化或多个连接
- **头部压缩**: HTTP/2 使用 HPACK 算法压缩头部，减少冗余传输
- **服务器推送**: HTTP/2 可主动推送资源，无需客户端显式请求
- **二进制帧**: HTTP/2 使用二进制格式替代文本格式，解析更高效
