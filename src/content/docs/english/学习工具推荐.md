## 1. 词汇记忆工具

### 1.1 Anki

| 项目 | 说明                                    |
| ---- | --------------------------------------- |
| 类型 | 间隔重复记忆软件                        |
| 平台 | Windows / macOS / Linux / iOS / Android |
| 价格 | 桌面端免费，iOS 付费                    |
| 官网 | https://apps.ankiweb.net                |

**核心功能：**

| 功能         | 说明                     |
| ------------ | ------------------------ |
| 间隔重复算法 | 基于遗忘曲线自动安排复习 |
| 自定义卡片   | 支持文字、图片、音频     |
| 同步功能     | 多设备同步学习进度       |
| 插件系统     | 丰富的社区插件           |
| 统计分析     | 学习数据可视化           |

**Anki 使用指南：**

1. **创建牌组 (Deck)**：按主题创建不同牌组
2. **添加笔记 (Note)**：添加单词、释义、例句
3. **学习模式**：新卡学习 → 复习 → 重新学习
4. **调整参数**：根据个人情况调整每日新卡和复习数量

**推荐参数设置：**

| 参数                | 推荐值   | 说明           |
| ------------------- | -------- | -------------- |
| New cards/day       | 15-30    | 每日新学卡片数 |
| Reviews/day         | 100-200  | 每日复习上限   |
| Learning steps      | 1m 10m   | 学习步骤间隔   |
| Graduating interval | 1 day    | 毕业间隔       |
| Easy interval       | 4 days   | 简单间隔       |
| Maximum interval    | 365 days | 最大间隔       |

**推荐插件：**

| 插件                     | 功能                 |
| ------------------------ | -------------------- |
| FastWordQuery            | 自动查询词典添加释义 |
| Image Occlusion Enhanced | 图片遮挡记忆         |
| Speed Focus Mode         | 限时复习             |
| Review Heatmap           | 学习热力图           |
| Edit Field During Review | 复习时编辑字段       |

## 7. 工具组合推荐

### 7.1 四六级备考组合

```
词汇：Anki + 欧路词典
语法：Grammarly + 语法书
阅读：沙拉查词 + VOA
听力：BBC Learning English
写作：Grammarly + Writefull
```

### 7.2 考研英语组合

```
词汇：Anki + 墨墨背单词
阅读：Zotero + 沙拉查词
翻译：DeepL + OmegaT
写作：Grammarly + Academic Phrasebank
真题：历年真题 + 错题本
```

### 7.3 技术英语组合

```
词汇：Anki（计算机专业词汇牌组）
阅读：DevDocs + 沙拉查词
写作：Grammarly + 技术文档模板
翻译：DeepL + 术语表
文档：Read the Docs + Markdown
```
