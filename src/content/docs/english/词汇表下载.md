## 6. 词汇表使用指南

### 6.1 导入 Anki

1. 下载 CSV 格式词汇表
2. 打开 Anki → File → Import
3. 选择 CSV 文件
4. 映射字段：单词 → Front，释义 → Back
5. 点击 Import

### 6.2 导入欧路词典

1. 下载 CSV 格式词汇表
2. 打开欧路词典 → 词汇表 → 导入
3. 选择 CSV 文件
4. 设置字段映射
5. 完成导入

### 6.3 自制词汇表

| 步骤 | 操作                 |
| ---- | -------------------- |
| 1    | 从阅读材料中收集生词 |
| 2    | 查词典获取音标和释义 |
| 3    | 从原文中提取例句     |
| 4    | 按主题或词频分类     |
| 5    | 定期复习和更新       |
