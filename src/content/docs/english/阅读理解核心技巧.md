## 1. 阅读的三大模式

### 1.1 三种阅读模式对比

| 模式 | 英文              | 目的         | 速度 | 精度 |
| ---- | ----------------- | ------------ | ---- | ---- |
| 略读 | Skimming          | 获取文章大意 | 最快 | 最低 |
| 扫读 | Scanning          | 定位特定信息 | 快   | 中   |
| 精读 | Intensive Reading | 深入理解细节 | 慢   | 最高 |

### 1.2 模式选择策略

```
阅读任务 → 先略读获取大意 → 扫读定位关键信息 → 精读理解深层含义
```

| 任务类型          | 推荐模式 |
| ----------------- | -------- |
| 了解文章主旨      | 略读     |
| 查找具体数据/人名 | 扫读     |
| 理解论证逻辑      | 精读     |
| 考试阅读理解      | 三者结合 |

## 2. 略读 (Skimming)

### 2.1 略读的目标

略读的目的是在**最短时间内**获取文章的**核心信息**：主题、结构、作者态度。

### 2.2 略读的步骤

1. **读标题和副标题** → 把握文章主题
2. **读首段** → 了解文章背景和主旨
3. **读每段首句** → 抓住段落主题句
4. **读末段** → 把握结论和作者态度
5. **浏览关键词** → 注意转折词、强调词

### 2.3 略读重点区域

| 区域   | 原因     | 关注内容       |
| ------ | -------- | -------------- |
| 标题   | 点明主题 | 核心概念       |
| 首段   | 引出主题 | 主题句、背景   |
| 段首句 | 段落主旨 | 主题句         |
| 段末句 | 段落结论 | 总结、过渡     |
| 末段   | 文章结论 | 作者观点、态度 |

### 2.4 略读时跳过的内容

- 具体数据和统计数字
- 详细举例和论证过程
- 插入语和括号内容
- 重复说明的部分

## 3. 扫读 (Scanning)

### 3.1 扫读的目标

扫读的目的是**快速定位**文章中的**特定信息**，如人名、日期、数字、术语等。

### 3.2 扫读的步骤

1. **明确目标** → 确定要查找的信息类型
2. **预判位置** → 根据文章结构推测信息可能出现的区域
3. **快速移动视线** → 以关键词为锚点快速扫视
4. **锁定目标** → 找到目标后精读相关句子

### 3.3 扫读的关键词类型

| 信息类型  | 视觉特征   | 示例                    |
| --------- | ---------- | ----------------------- |
| 人名      | 大写字母   | John Smith, MIT         |
| 日期/数字 | 阿拉伯数字 | 2026, 3.14, 50%         |
| 术语      | 斜体/加粗  | _algorithm_, **AI**     |
| 地名      | 大写字母   | Silicon Valley, Beijing |
| 特殊格式  | 引号/括号  | "deep learning", (CPU)  |

### 3.4 扫读技巧

- **Z 字形扫视**：视线从左上到右上，再从右下到左下
- **关键词锚定**：以视觉突出的词为锚点
- **区域定位**：根据文章结构缩小搜索范围

## 4. 精读 (Intensive Reading)

### 4.1 精读的目标

精读的目的是**深入理解**文章的**细节、逻辑和深层含义**。

### 4.2 精读的层次

| 层次     | 目标         | 方法               |
| -------- | ------------ | ------------------ |
| 字面理解 | 理解字面意思 | 逐句阅读，查词典   |
| 推断理解 | 理解言外之意 | 分析暗示和隐含意义 |
| 评价理解 | 评价作者观点 | 判断论据是否充分   |
| 应用理解 | 应用文章内容 | 将信息迁移到新情境 |

### 4.3 精读的核心技能

**1) 识别主题句 (Topic Sentence)**

> 主题句通常出现在段首，有时在段末，偶尔在段中。

**2) 理解逻辑关系 (Logical Relationships)**

| 关系类型 | 标志词                                               |
| -------- | ---------------------------------------------------- |
| 因果     | because, since, therefore, consequently, as a result |
| 转折     | however, nevertheless, yet, but, on the contrary     |
| 递进     | moreover, furthermore, in addition, additionally     |
| 举例     | for example, for instance, such as, namely           |
| 对比     | while, whereas, in contrast, on the other hand       |
| 总结     | in conclusion, to sum up, overall, in brief          |
| 条件     | if, provided that, unless, assuming that             |

**3) 理解指代关系 (Reference)**

| 代词                  | 可能指代             |
| --------------------- | -------------------- |
| it/they               | 前文提到的名词       |
| this/that             | 前文提到的概念或句子 |
| such                  | 前文提到的同类事物   |
| the former/the latter | 前文提到的两者之一   |

**4) 理解修辞手法 (Rhetorical Devices)**

| 修辞 | 功能                 | 示例                                      |
| ---- | -------------------- | ----------------------------------------- |
| 比喻 | 使抽象概念具体化     | The internet is an information highway.   |
| 反讽 | 表达与字面相反的意思 | What a wonderful day! (when it's raining) |
| 夸张 | 强调观点             | I've told you a million times.            |
| 设问 | 引起思考             | Is AI a friend or foe?                    |

## 5. 阅读速度提升

### 5.1 影响阅读速度的因素

| 因素       | 影响       | 改善方法                   |
| ---------- | ---------- | -------------------------- |
| 词汇量不足 | 频繁查词   | 扩大词汇量，根据上下文猜词 |
| 逐词阅读   | 速度慢     | 练习意群阅读               |
| 回读       | 重复阅读   | 用手指引导视线             |
| 心读       | 默念每个词 | 练习视觉阅读               |
| 注意力分散 | 效率低     | 定时训练，集中注意力       |

### 5.2 意群阅读法

意群 (Sense Group) 是有意义的词组单位，以意群为单位阅读可大幅提升速度：

> 逐词：The / algorithm / was / designed / to / optimize / the / performance / of / the / system.
> 意群：The algorithm / was designed / to optimize / the performance / of the system.

### 5.3 阅读速度标准

| 等级     | 速度 (wpm) | 适用场景     |
| -------- | ---------- | ------------ |
| 初级     | 100-150    | 精读学术文章 |
| 中级     | 150-250    | 考试阅读理解 |
| 高级     | 250-400    | 日常英文阅读 |
| 母语水平 | 400-600    | 快速浏览     |

## 6. 考试阅读策略

### 6.1 先读题 vs. 先读文

| 策略         | 优点             | 缺点             | 适用       |
| ------------ | ---------------- | ---------------- | ---------- |
| 先读题后读文 | 目标明确，效率高 | 可能缺乏整体理解 | 细节题为主 |
| 先读文后读题 | 整体理解好       | 可能遗忘细节     | 主旨题为主 |
| 边读边做题   | 平衡效率和理解   | 可能打断阅读节奏 | 时间充裕   |

### 6.2 时间分配

| 考试类型 | 总时间    | 每篇时间  | 建议分配                |
| -------- | --------- | --------- | ----------------------- |
| 四级     | 40 min    | 8-10 min  | 读文3min + 做题5-7min   |
| 六级     | 40 min    | 8-10 min  | 读文3min + 做题5-7min   |
| 考研     | 60-70 min | 15-20 min | 读文5min + 做题10-15min |

### 6.3 答题顺序

1. 先做**主旨题**和**态度题**（需要整体理解）
2. 再做**细节题**和**词义题**（需要定位信息）
3. 最后做**推断题**（需要综合分析）

## 7. 日常阅读训练

### 7.1 分级阅读材料

| 级别 | 材料                           | 特点             |
| ---- | ------------------------------ | ---------------- |
| 入门 | Graded Readers, 简写本         | 词汇量 1000-2000 |
| 初级 | 新闻简讯 (VOA Special English) | 词汇量 2000-3000 |
| 中级 | The Economist 精选, 科技博客   | 词汇量 4000-6000 |
| 高级 | 学术论文, Nature/Science       | 词汇量 8000+     |

### 7.2 阅读习惯养成

- **每日定量**：每天至少阅读 15-30 分钟英文材料
- **主题阅读**：选择感兴趣的主题深入阅读
- **主动阅读**：边读边做笔记、提问题
- **复盘总结**：读完后用英文写简短摘要
