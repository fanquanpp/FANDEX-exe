## 1. 题型分类总览

### 1.1 四大题型

| 题型   | 占比   | 难度  | 核心能力 |
| ------ | ------ | ----- | -------- |
| 主旨题 | 15-20% | 中    | 概括归纳 |
| 细节题 | 40-50% | 低-中 | 定位提取 |
| 推断题 | 20-25% | 高    | 逻辑推理 |
| 词义题 | 10-15% | 中    | 语境推断 |

## 2. 主旨题 (Main Idea Questions)

### 2.1 题型识别

| 常见问法                                            |
| --------------------------------------------------- |
| What is the main idea of the passage?               |
| What is the passage mainly about?                   |
| Which of the following best summarizes the passage? |
| The author's primary purpose is to...               |
| What is the best title for this passage?            |

### 2.2 解题策略

**策略一：找主题句**

| 位置 | 概率   | 标志         |
| ---- | ------ | ------------ |
| 段首 | 60-70% | 首句即主题句 |
| 段末 | 15-20% | 总结性语句   |
| 段中 | 10-15% | 转折后引出   |

**策略二：排除干扰项**

| 干扰类型 | 特征           | 示例                                 |
| -------- | -------------- | ------------------------------------ |
| 以偏概全 | 只涉及部分内容 | 文章讲AI整体，选项只提机器学习       |
| 过于宽泛 | 超出文章范围   | 文章讲深度学习，选项提整个计算机科学 |
| 无中生有 | 文章未提及     | 文章未讨论的内容                     |
| 与文相反 | 与作者观点矛盾 | 与文章主旨相反的表述                 |

**策略三：归纳法**

当文章没有明确的主题句时，需要从各段内容中归纳：

1. 提取每段的关键词
2. 找出各段关键词的共性
3. 用一句话概括这个共性

### 2.3 实战示例

> **文章摘要：** 第一段介绍全球变暖的现状；第二段分析全球变暖的原因；第三段讨论全球变暖的影响；第四段提出应对措施。

**Q: What is the main idea?**

| 选项                                   | 判断 | 理由                     |
| -------------------------------------- | ---- | ------------------------ |
| A. The causes of global warming        | ❌   | 以偏概全（只涉及第二段） |
| B. Global warming and its implications | ✅   | 涵盖原因、影响和对策     |
| C. Environmental protection            | ❌   | 过于宽泛                 |
| D. How to reduce carbon emissions      | ❌   | 以偏概全（只涉及第四段） |

## 3. 细节题 (Detail Questions)

### 3.1 题型识别

| 常见问法                                     |
| -------------------------------------------- |
| According to the passage, ...                |
| Which of the following is true/NOT true?     |
| The author mentions X in order to...         |
| All of the following are mentioned EXCEPT... |
| According to paragraph X, ...                |

### 3.2 解题策略

**策略一：关键词定位法**

1. 在题干中找出**关键词**（人名、数字、术语、特殊表达）
2. 在文章中**快速定位**关键词
3. 精读关键词所在的**句子及前后句**
4. 与选项进行**比对**

**策略二：同义替换识别**

正确选项通常是原文的**同义替换**，而非原文照搬：

| 原文        | 同义替换                        |
| ----------- | ------------------------------- |
| significant | substantial, considerable       |
| increase    | rise, grow, surge               |
| decrease    | decline, reduce, diminish       |
| cause       | lead to, result in, bring about |
| important   | crucial, vital, essential       |
| problem     | issue, challenge, difficulty    |

**策略三：排除法**

| 排除标准   | 说明                 |
| ---------- | -------------------- |
| 与原文矛盾 | 直接排除             |
| 原文未提及 | 直接排除             |
| 过度推断   | 超出原文信息范围     |
| 偷换概念   | 改变原文的主语或范围 |

### 3.3 细节题的陷阱

| 陷阱类型 | 特征             | 示例                                     |
| -------- | ---------------- | ---------------------------------------- |
| 偷换概念 | 改变主语或修饰语 | 原文：most students → 选项：all students |
| 时态错误 | 改变时间信息     | 原文：has developed → 选项：will develop |
| 因果颠倒 | 颠倒因果关系     | 原文：A causes B → 选项：B causes A      |
| 绝对化   | 将相对变为绝对   | 原文：may help → 选项：definitely cures  |
| 张冠李戴 | 混淆不同对象     | 原文：A did X, B did Y → 选项：A did Y   |

### 3.4 NOT/EXCEPT 题型

这类题需要找出**与原文不符**的选项：

1. 先处理能确定的选项
2. 逐个与原文比对
3. 剩余无法在原文找到对应的就是答案

## 4. 推断题 (Inference Questions)

### 4.1 题型识别

| 常见问法                                                  |
| --------------------------------------------------------- |
| It can be inferred from the passage that...               |
| What does the author imply about...?                      |
| The passage suggests that...                              |
| Which of the following can be concluded from the passage? |
| The author would most likely agree that...                |

### 4.2 解题原则

**核心原则：推断 ≠ 臆测**

- ✅ 基于原文信息的合理推导
- ❌ 脱离原文的主观猜测
- ✅ 答案在原文中有依据
- ❌ 答案需要额外知识

### 4.3 推断题类型

**1) 逻辑推断**

根据原文的因果关系进行推断：

> 原文：The new policy reduced carbon emissions by 30%.
> 推断：The policy was effective in addressing environmental concerns.

**2) 态度推断**

根据作者的用词和论述方式推断态度：

| 用词倾向                        | 态度        |
| ------------------------------- | ----------- |
| clearly, undoubtedly, certainly | 肯定/自信   |
| perhaps, possibly, might        | 不确定/谨慎 |
| unfortunately, regrettably      | 遗憾/批评   |
| surprisingly, unexpectedly      | 惊讶        |
| importantly, significantly      | 强调/重视   |

**3) 预测推断**

根据文章趋势预测未来发展：

> 原文：The trend toward remote work has accelerated since 2020.
> 推断：Remote work will likely continue to grow in the future.

**4) 反向推断**

根据原文的正面表述推断反面：

> 原文：Only students who complete all assignments can take the final exam.
> 推断：Students who don't complete all assignments cannot take the final exam.

### 4.4 推断题的解题步骤

1. 精读题干，明确推断对象
2. 定位原文相关段落
3. 分析原文的逻辑关系
4. 基于原文进行合理推导
5. 验证推断是否在原文有依据

## 5. 词义题 (Vocabulary Questions)

### 5.1 题型识别

| 常见问法                                       |
| ---------------------------------------------- |
| The word "X" in line Y most probably means...  |
| What does the author mean by "X"?              |
| The phrase "X" is closest in meaning to...     |
| The word "X" in the passage is used to mean... |

### 5.2 解题策略

**策略一：语境推断法**

利用上下文线索推断词义：

| 线索类型 | 标志                            | 示例                                                   |
| -------- | ------------------------------- | ------------------------------------------------------ |
| 定义线索 | is defined as, means, refers to | Algorithm **refers to** a step-by-step procedure.      |
| 重述线索 | that is, in other words, or     | The result was **anomalous**, **that is**, unusual.    |
| 举例线索 | such as, for example, like      | **Herbivores**, **such as** cows and deer, eat plants. |
| 对比线索 | however, but, unlike, whereas   | Unlike the **gregarious** sister, she was solitary.    |
| 因果线索 | because, therefore, thus        | The **arid** climate **caused** the crops to wither.   |
| 同义线索 | and, or, similarly              | The **ubiquitous** and **omnipresent** smartphones...  |

**策略二：构词法分析**

| 构词法 | 示例            | 分析                                        |
| ------ | --------------- | ------------------------------------------- |
| 前缀   | unpredictable   | un- (否定) + predict (预测) + -able (可…的) |
| 后缀   | standardization | standard + -ize + -ation (名词化)           |
| 合成   | breakthrough    | break + through                             |
| 转化   | to google       | Google (名词) → google (动词)               |

**策略三：代入验证法**

1. 将各选项代入原文
2. 检查句意是否通顺
3. 检查是否与上下文逻辑一致

### 5.3 词义题注意事项

| 注意点             | 说明                               |
| ------------------ | ---------------------------------- |
| 不要选熟词僻义     | 题目考的通常是该词在文中的特定含义 |
| 注意上下文         | 同一个词在不同语境中含义不同       |
| 排除过于简单的选项 | 考研/六级词义题通常考较难含义      |
| 注意感情色彩       | 词义应与上下文感情色彩一致         |

## 6. 综合解题流程

### 6.1 标准解题流程

```
1. 略读全文 → 把握主旨和结构（2-3分钟）
2. 阅读题干 → 确定题型和关键词
3. 定位原文 → 找到相关信息
4. 精读分析 → 深入理解相关段落
5. 比对选项 → 排除干扰，选择答案
6. 验证答案 → 回到原文确认
```

### 6.2 时间管理

| 题型   | 建议用时  | 策略           |
| ------ | --------- | -------------- |
| 主旨题 | 1-2 min   | 先做，基于略读 |
| 细节题 | 1-1.5 min | 定位+比对      |
| 推断题 | 1.5-2 min | 深入分析       |
| 词义题 | 0.5-1 min | 语境推断       |

### 6.3 常见错误总结

| 错误类型 | 原因               | 对策                      |
| -------- | ------------------ | ------------------------- |
| 定位错误 | 关键词选错         | 选最独特的词定位          |
| 过度推断 | 超出原文范围       | 答案必须有原文依据        |
| 忽视转折 | 没看到 however/but | 注意转折词                |
| 时间不够 | 某题耗时过长       | 控制每题时间              |
| 粗心大意 | 看错题干要求       | 仔细审题，注意 NOT/EXCEPT |
